# 🌍 India Air Pollution AI Dashboard

## 📌 Project Overview

This project is a Machine Learning-based system that predicts Air Quality Index (AQI) and visualizes pollution data across different cities in India.

It combines:

* Machine Learning (Random Forest)
* Data Analysis
* Interactive Dashboard (Streamlit)
* Live AQI API integration

---

## 🚀 Features

* 📊 AQI Prediction using ML model
* 🌆 City-based AQI selection
* 🌐 Live AQI data using API
* 🗺️ Interactive India pollution map
* 📈 Pollution contribution charts
* 📉 AI-based 7-day AQI forecast

---

## 🧠 Machine Learning Model

* Algorithm: Random Forest Regressor
* Accuracy: ~89% (R² Score)
* Input Features:

  * PM2.5
  * PM10
  * NO2
  * CO
  * O3
